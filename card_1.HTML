Project Overview

Ki is my most ambitious personal project to date. It started as a rough concept sketch and slowly turned into a fully modelled and textured game asset over the course of four months.

The idea was to design a modular combat mech that could work in both a sci-fi military game and a darker, post-apocalyptic setting. Every armour plate is a separate object, which makes it easy to swap pieces and customise the look.

I used Blender for all modelling, rigging, and rendering. Texturing was done in Substance Painter using a PBR metalness workflow. The final render was lit with three-point cinematic lighting and a subtle rim light to give the mech its dramatic silhouette.

One of the biggest challenges was keeping the polygon count low enough for real-time use while still having enough surface detail to look convincing in close-up shots. The final mesh sits at around ~85,000 triangles, which is reasonable for a hero character asset.

This project taught me a lot about hard-surface topology, normal map baking, and how to plan a model from the start so that rigging doesn't become a nightmare later on.

Tools Used

  • Substance Painter
